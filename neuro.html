GitHub Sync Settings

Neuro KB saves your pearls straight to neuro-data.json in your My-Tools repo, so they show up on every device. To write to GitHub, this page needs a Personal Access Token — it's stored only in this browser's local storage, never uploaded anywhere else.

Create one at github.com/settings/personal-access-tokens/new → Repository access: Only select repositories → My-Tools → Permissions: Contents → Read and write.
